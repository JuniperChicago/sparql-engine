    • Defined in operators/optional.ts:67

    Handles an SPARQL OPTIONAL clause in the following way: 1) Buffer every bindings produced by the source iterator. 2) Set a flag to False if the OPTIONAL clause yield at least one set of bindings. 3) When the OPTIONAL clause if evaluated, check if the flag is True. 4) If the flag is True, then we output all buffered values.
